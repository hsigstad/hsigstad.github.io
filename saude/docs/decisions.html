2026-02-09 — Adopted two-tier empirical design with individual-level data

Decision: Restructure the empirical strategy around a two-tier design: (1) municipality-period population-level analysis as the spine, (2) litigant-level case-by-case analysis as a second pillar. Upgrade from municipality-year to municipality-month/quarter where feasible. Add two-endogenous-variable model (judicial vs administrative spending). Construct non-litigant outcomes by excluding linked litigants from SIH/SIM.

Reason:

  • Individual-level microdata (SIH, SIM, RAIS, name registry with ~50% unique names) now available on secure server
  • Two-tier design directly addresses multiple referee concerns: power for health outcomes, distributional incidence methodology, exclusion restriction (decomposition shows what drives aggregate effect), and monotonicity (case-level tests possible)
  • Municipality-month increases power substantially over municipality-year
  • Two-endogenous-variable model addresses the "two LATEs" critique of the judicial vs administrative spending comparison
  • Non-litigant outcome construction provides direct crowd-out evidence
  • Decomposition (τ ≈ ω τ^L + (1-ω) τ^NL) disciplnes interpretation of aggregate effect

Alternatives considered:

  • Keep municipality-year only (rejected: leaves power concerns unaddressed; wastes microdata granularity)
  • Individual-level only (rejected: municipality-level fiscal mechanism is the paper's distinctive contribution)
  • Three-tier design with neighborhood/catchment area (rejected: scope creep; insufficient geographic precision in SIH)

Implications:

  • Paper Sections 4–9 restructured to reflect two-tier design
  • Need to identify clean instrument for administrative spending (transfer formula threshold/revision)
  • Need to implement name-linkage with IPW
  • Need to build non-litigant outcome aggregation pipeline
  • Scope risk: keep litigant-level to one health index + one labor outcome in main text; rest in appendix
