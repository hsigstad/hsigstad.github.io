Municipal Spending

TCE-SP line-item municipal expenditure data with judicial spending markers.

Overview

TCE-SP (Tribunal de Contas do Estado de São Paulo) is the state audit court that oversees municipal finances in São Paulo. Data comes from the Audesp (Auditoria Eletrônica de Órgãos Públicos) system, which collects detailed fiscal information from all municipalities.

The spending data covers 644 municipalities in SP state (excluding the São Paulo capital municipality, which is audited by TCM-SP). Each row represents an individual budget execution event — an empenho (commitment), liquidação (verification), or pagamento (payment).

Source: transparencia.tce.sp.gov.br/conjunto-de-dados

Despesas Structure

The Despesas (expenditure) dataset is the core data source. Key columns include:

ColumnDescription
cd_municipioMunicipality code
ds_municipioMunicipality name
dt_emissao_despesaCommitment (empenho) issue date
vl_despesaExpenditure amount (R$)
ds_funcao_governoGovernment function — filter to SAÚDE for health spending
ds_subfuncao_governoSub-function — enables crowd-out analysis across health sub-areas
ds_modalidade_licProcurement modality
historico_despesaFree-text description — contains judicial markers for identifying court-mandated spending
tp_despesaExpenditure type: Empenhado, Liquidado, or Pago
ds_elementoBudget element classification
ds_fonte_recursoFunding source

The budget execution cycle follows three stages: Empenhado (committed) → Liquidado (verified/received) → Pago (paid). Each stage generates a separate row.

Judicial Spending Identification

Judicial mandates can be identified via keyword patterns in the historico_despesa free-text field (e.g., "judicial", "mandado", "liminar", "tutela antecipada"). The share of health spending attributable to court orders varies substantially across municipalities:

  • Many small municipalities: 0.0% judicial health spending
  • Medium municipalities: typically 1–3%
  • High-litigation municipalities: up to 6.8%

This variation across municipalities — driven by differences in judicial behavior — is the key source of identifying variation for the project's fiscal analysis.
