BEC state procurement — consolidated (Licitações do Estado de SP)

Source

  • provider: BEC/SP (Bolsa Eletrônica de Compras) — São Paulo state's electronic procurement platform
  • access: data/Licitações do Estado de SP/
  • documentation:

Files

  • BEC_consolidated.csv — ~4M rows, 83 columns (semicolon-delimited)
    • one row per item per purchase order (OC)
    • columns include: purchase order details, item descriptions, buyer unit, supplier info, bid counts by phase, prices, firm characteristics (CNPJ, CNAE, size, location, distance to buyer)
  • BEC_bid_to_bid.csv — ~2.3M rows, 52 columns
    • one row per bid per item per auction
    • columns include: auction dates, item details, unit price proposed, negotiated price, reference price, winner flag, item quantities

Coverage

  • time period: (sample — full universe for recent period forthcoming)
  • geographic scope: São Paulo state (state-level purchases, not municipal)
  • population: state procurement via BEC electronic platform

Unit of observation

  • consolidated: item × purchase order (OC)
  • bid_to_bid: bid × item × auction

Key variables

  • numerodaoc — purchase order number
  • proc_compra — procurement procedure type
  • preco_final — final price
  • numbids_ph1–ph7 — number of bids by auction phase
  • firm_cnpj — supplier CNPJ
  • cnae_fiscal / cnae_descr — supplier industry classification
  • porte_empresa — firm size
  • dist / dist1 / dist2 / dist3 — distance measures (buyer–supplier)
  • same_municip — buyer and supplier in same municipality
  • Flag Vencedor (bid_to_bid) — winner indicator
  • Valor Unitário Proposta / Valor Unitario Negociado (bid_to_bid) — bid and negotiated prices

Sample restrictions

  • state-level purchases only (not municipal)

Data quality

  • filenames encode procedure IDs, e.g. 080102000012011OC00097_edital.txt
  • content: HTML-sourced text from BEC/SP system ("Sistema BEC/SP")

Coverage

  • time period: (sample — full set forthcoming)
  • geographic scope: São Paulo state
  • population: state public bids

Unit of observation

  • one file per public notice (edital)

Key variables

  • procedure ID (ordem de compra) — encoded in filename

Judicial case files (example_procurement_cases.csv)

Source

  • provider: TJSP (Tribunal de Justiça do Estado de São Paulo)
  • access: data/example_procurement_cases.csv
  • documentation:

Files

  • example_procurement_cases.csv — ~3,700 rows, ~130 MB

Coverage

  • time period: various (cases with decisions from ~2013 onwards in sample)
  • geographic scope: São Paulo state
  • population: procurement-related judicial cases (improbidade administrativa, ação civil pública, etc.)

Unit of observation

  • one row per case

Key variables

  • num_npu — CNJ unified case number
  • classe — case type (e.g., Ação Civil Pública)
  • assunto — subject (e.g., Improbidade Administrativa)
  • magistrado — judge name
  • comarca / foro / vara — court jurisdiction
  • data — decision date
  • text — full decision text (sentença)

Sample restrictions

  • sample of ~3,700 cases (full universe forthcoming)

Data quality

  • full text available for LLM-based scheme classification

Municipal procurement (TCE-SP — Licitações Municípios Estado de SP)

Source

  • provider: TCE-SP (Tribunal de Contas do Estado de São Paulo) — Portal da Transparência Municipal
  • URL: https://transparencia.tce.sp.gov.br/conjunto-de-dados
  • access: data/Licitações Municípios Estado de SP/
  • documentation: Audesp system; see also saude project docs/data.md for detailed TCE-SP documentation

Files

  • licitacao-2024-11_0.zip — 47 MB (November 2024)
  • licitacao-2024-12_0.zip — 49 MB (December 2024)
  • despesas-2012.zip — 976 MB (municipal expenditures 2012)

Coverage

  • time period: sample months (full series 2018+ forthcoming for licitações; 2008–2024 for despesas)
  • geographic scope: 644 municipalities in São Paulo state (capital excluded — falls under TCM-SP)
  • population: municipal procurement and expenditures

Unit of observation

  • licitações: procurement event level
  • despesas: line-item expenditure (empenho / liquidação / pagamento)

Key variables (despesas)

  • ds_municipio / codigo_municipio_ibge — municipality
  • tp_despesa — type: Empenhado / Liquidado / Pago
  • vl_despesa — value in R$
  • ds_modalidade_lic — procurement modality (PREGÃO, DISPENSA, etc.)
  • historico_despesa — free-text description (can contain judicial markers)
  • ds_funcao_governo / ds_subfuncao_governo — functional classification

Sample restrictions

  • sample months only (full universe forthcoming)

Data quality

  • encoding: ISO-8859-1 (not UTF-8)
